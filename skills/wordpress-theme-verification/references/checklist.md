# Static checklist

- Target is a readable directory.
- Root `style.css` contains a non-empty `Theme Name`.
- Classic evidence: root `index.php`; block evidence: `templates/index.html`; both: hybrid.
- Present PHP can be syntax-checked only when a local PHP executable is explicitly available.
- Present `theme.json` must parse as JSON.
- Do not extrapolate static findings into activation, rendering, compatibility, visual, or performance claims.
