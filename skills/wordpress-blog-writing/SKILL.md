# WordPress blog writing

Produce title, optional slug, meta description, categories, tags, introduction, headings, useful body, conclusion, and optional CTA. Do not invent sources, statistics, testimonials, prices, or capabilities. Mark unsupported factual claims as `[RESEARCH NEEDED]`.
