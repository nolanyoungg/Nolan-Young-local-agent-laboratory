# Facebook Page publishing

Generate a concise caption from the supplied title only. Dry runs never contact Meta. A publish requires configured Page ID, `--approve --publish`, and `--confirm` equal to that Page ID. Use the Page photo endpoint only; do not automate facebook.com or personal profiles.
